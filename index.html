Farcaster Hub Feeds Easily generate RSS, Atom, and JSON feeds directly from a Farcaster Hub. How To Use All feeds exclude replies and are sorted in reverse chronological order. /rss/user/{fid} Generate an RSS feed of a user's casts. /atom/user/{fid} Generate an Atom feed of a user's casts. /json/user/{fid} Generate an JSON feed of a user's casts. Configuration To use a specific Farcaster Hub, add the "hub" query parameter where the value includes the HTTP port (eg: https://hub.freefarcasterhub.com:3281)
